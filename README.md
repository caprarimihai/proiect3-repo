# Proiect Java - Probleme mici

## Descriere
Acest proiect conține 3 aplicații simple realizate în Java, fiecare rezolvând o problemă diferită de bază.

---

## Structura proiectului

- `Anotimpuri.java` – determină anotimpul în funcție de lună
- `JocAria.java` – calculează aria unor figuri geometrice
- `MeniuPizzaMania.java` – meniu simplu de comandă pizza

---

## Tehnologii folosite
- Java (JDK 17+)
- Console application
- Git pentru versionare

---

### 1. Compilează fișierul
